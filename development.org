** Documentation

Durable documentation follows the path-first Johnny.Decimal Org topology under
[[file:docs/index.org][docs/index.org]] and
[[file:docs/topology.toml][docs/topology.toml]].

Use `docs/AC_slug/AC.ID_semantic_name.org` for long-lived records. The
category prefix in the directory and the coordinate in the file name must
agree. If a new category is needed, update `docs/topology.toml` explicitly
instead of silently inventing category meaning.

#+begin_src sh
direnv exec . cargo run --quiet -- lint docs development.org
#+end_src
